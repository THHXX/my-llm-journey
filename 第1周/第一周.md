# 第1周学习笔记

## 环境与 Git
- 已使用 Python 3.10+、Git、VS Code。
- 基本 Git 流程：`git status` → `git add <files>` → `git commit -m "message"` → `git push origin main`。
  - 示例提交信息：`chore: add hello LLM script`（杂项），`feat: add word frequency counter script`（新增功能）。

## Python 基础要点
- 变量与类型：`int`/`float`/`str`/`bool`，容器 `list`/`tuple`/`set`/`dict`。
- 条件与循环：`if/elif/else`，`for` 遍历列表/字典，`while` 计数，`range(start, stop, step)`。
- 函数：默认参数与返回值示例：
  ```python
  def greet(name, prefix="Hello"):
      return f"{prefix}, {name}!"
  ```
- 类与对象：
  ```python
  class Student:
      def __init__(self, name, score):
          self.name = name
          self.score = score

      def is_pass(self):
          return self.score >= 60
  ```
- 文件读写：
  ```python
  from pathlib import Path

  with Path("demo.txt").open("w", encoding="utf-8") as f:
      f.write("hello")
  with Path("demo.txt").open("r", encoding="utf-8") as f:
      content = f.read()
  ```

## 词频统计脚本（`word_count/word_count.py`）
- 主要步骤：读取文本 → 小写化与去标点 → `split` 分词 → 字典计数 → 输出 Top N → 写入结果文件。
- 运行示例（在仓库根目录）：
  ```bash
  python word_count/word_count.py
  python word_count/word_count.py --input word_count/input.txt --output word_count/output.txt --top 5
  ```
- 关键函数：
  ```python
  def clean_text(text: str) -> str:
      lowered = text.lower()
      for ch in PUNCTUATION:
          lowered = lowered.replace(ch, " ")
      return lowered

  def count_words(words: Iterable[str]) -> Dict[str, int]:
      freq = {}
      for w in words:
          freq[w] = freq.get(w, 0) + 1
      return freq
  ```
- 输入输出：默认读取 `word_count/input.txt`，生成 `word_count/output.txt`，终端打印 Top N 与总词数/唯一词数。
