## 第一步：完成安装 (最关键)

你之前的终端卡在 `Create a default Unix user account:`，这是最后一步。

1.  **回到那个黑色窗口** (或者重新在开始菜单搜索打开 `Ubuntu`)。
2.  **输入用户名**：输入 `user` (或者你喜欢的英文名)，按 **回车**。
3.  **输入密码**：
    *   随便想一个简单密码（比如 `123456`）。
    *   **注意**：输入时**屏幕不会显示任何东西**，这是正常的！
    *   输完按 **回车**。
    *   再输一次确认，按 **回车**。
4.  **成功标志**：
    当看到绿色的文字 `user@你的电脑名:~$` 时，恭喜你，你已经进入 Linux 世界了！

---

## 第二步：连接 VS Code (神器)

不要在那个黑色窗口里写代码，我们要用 VS Code "遥控" 它。

1.  **安装插件**：
    *   点击 VS Code 左侧的 **扩展图标** (四个方块)。
    *   搜索 `WSL`。
    *   安装由 **Microsoft** 开发的 `WSL` 插件。

2.  **一键穿越**：
    *   在那个**黑色窗口 (Ubuntu 终端)** 里，输入以下命令并回车：
        ```bash
        cd /mnt/f/Desktop/个人资料/大模型学习项目/第40周
        code .
        ```
    *   **神奇的事情发生了**：VS Code 会打开一个新的窗口，左下角显示绿色 `WSL: Ubuntu`。
    *   **现在，你在这个新窗口里做的所有操作，实际上都是在 Linux 里进行的！**

---

## 第三步：配置环境

在新的 VS Code 窗口中，打开顶部菜单 **终端 -> 新建终端** (Terminal)。你会发现终端变成了 Linux 风格。

依次复制并运行以下命令 (中间可能需要输入你的密码 `123456`)：

1.  **更新系统软件库** (相当于刷新应用商店)：
    ```bash
    sudo apt update
    ```

2.  **安装 Python 和 pip**：
    ```bash
    sudo apt install python3 python3-pip -y
    ```

3.  **安装 vLLM** (主角登场)：
    ```bash
    pip install vllm
    ```
    *(这一步可能需要几分钟，如果下载慢，请告诉我，我教你换源)*

---

## 第四步：运行 vLLM

环境装好了，现在验证成果！

1.  **运行脚本**：
    在 VS Code 终端中输入：
    ```bash
    python3 vllm_test.py
    ```

2.  **观察结果**：
    如果你看到 `🚀 正在使用 vLLM 加载模型...`，说明你成功了！你已经做到了 90% Windows 用户做不到的事情。

## 常见问题

*   **Q: 怎么访问 Windows 的文件？**
    *   A: 所有盘符都在 `/mnt/` 下。C盘是 `/mnt/c`，F盘是 `/mnt/f`。
*   **Q: 报错 `ModuleNotFoundError`?**
    *   A: 记得是用 `pip install` 安装，运行用 `python3` (Linux 默认 python 是 2.x，要用 python3)。
