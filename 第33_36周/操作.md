## 1. 项目简介
本项目是一个**全栈式智能客服系统**。
- **核心大脑**：基于 `Qwen-Max` 大模型。
- **知识库**：基于 RAG 技术，外挂 `data/` 目录下的企业文档（如退换货政策）。
- **交互界面**：使用 `Gradio` 构建美观的 Web 聊天窗口。
- **国产化**：完全使用国内服务（阿里云 DashScope + 魔搭 ModelScope），无需科学上网。

## 2. 环境安装
请在终端中运行以下命令安装依赖：

```powershell
# 1. 进入本周目录
cd "f:\Desktop\个人资料\大模型学习项目\第33_36周"

# 2. 安装依赖包
pip install -r requirements.txt
```

> **注意**：如果安装速度慢，可以使用清华源：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`

## 3. 本地运行
确保你已经安装好依赖，且 `DASHSCOPE_API_KEY` 环境变量已设置。

1. **运行应用**：
   ```powershell
   python app.py
   ```

2. **访问界面**：
   终端会显示类似 `Running on local URL:  http://127.0.0.1:7860` 的链接。
   按住 `Ctrl` 点击链接，或在浏览器输入该地址即可使用。

3. **测试对话**：
   - 试着问：“多少钱包邮？”（它会根据 `data/faq.txt` 回答满199包邮）
   - 试着问：“如果不喜欢能退吗？”（它会回答支持七天无理由）

## 4. 部署至 ModelScope (可选)
如果你想把这个系统发布给别人用，可以部署到阿里魔搭社区（免费）。

1. 注册并登录 [ModelScope 魔搭社区](https://www.modelscope.cn/)。
2. 点击顶部 **“创空间”** -> **“创建创空间”**。
3. 填写信息：
   - 英文名：`smart-cs-bot`
   - 接入 SDK：选择 **Gradio**
4. 创建成功后，网页上会有一个 Git 地址。
5. 将我们本地的代码（`app.py`, `rag_engine.py`, `requirements.txt`, `data/`）上传到那个 Git 仓库即可。
